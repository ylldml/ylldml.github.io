linux学习笔记

发表于2022-03-01|更新于2022-03-04
|阅读量:

1. whatis

显示在线手册说明

  • print name of current/working directory 打印当前/工作目录的名称
    list directory contents 列出目录的内容

2. ls

列出目录的内容

3. cd

改变工作目录

  • cd test 进入test目录
  • cd … 返回上级目录

4. cp

copy files 复制文件

  • cp hello.c hello2.txt （cp 复制文件 重命名该文件）
  • cp -r csource csource001 (cp 复制文件夹 重命名该文件夹)

5. less

查看文件

6. head tail

查看文件 的开头或结尾

  • head -n 1 hello.c
    • 查看hello.c文件里的第一行内容
  • tail -n 1 hello.c
    • 查看hello.c文件里的倒数第一行内容

7. find

  • 查找过滤文件

在目录层次结构中搜索文件

8. cat

合并多个文件夹

查看文件夹里的内容

9. cal

显示当前日历

10. which

定位文件的存放路径

11. whereis

定位程序，帮助手册的存放路径

12. touch

更改文件的时间戳

13. top

显示Linux进程

14. rm

删除文件或文件夹

15. mv

剪切文件或文件夹

16. mkdir

创建目录
