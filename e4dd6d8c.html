C语言基础

变量

  • 内存中的一小块空间

标识符命名规范

  • 标识符可以由字母、数字、“_”、“$” 组成。
  1. 标识符不能以数字开头。
  2. 标识符不能是 Java 的关键字。

scanf 函数

常量

#define 指令

const 关键字

二进制

计算机中的数据单位

  • 1B = 8bit
  • 1KB = 1024B
  • 1MB = 1024KB
  • 1GB = 1024MB

原码、反码、补码

  • 原码
    • 最高位存储符号（0 - 正，1 - 负），其他位存数据的绝对值。
      如：[-2] 原 = 1000 0010
  • 反码
    • [正数] 反 = [正数] 原 ，[负数] 反 = [负数] 原 除符号位外按位取反。
      如：[-2] 反 = 1111 1101
  • 补码
    • [正数] 补 = [正数] 原 ，[负数] 补 = [负数] 反 + 1。
      如：[-2] 补 = 1111 1110

基本数据类型

递增与递减运算符

用法

  • 前缀模式: 将符号写在操作数之前(++i), 使其先递增/减, 再参与其他运算。
  • 后缀模式: 将符号写在操作数之后(i++), 使其先参与其他运算, 再递增/减。

break和continue

猜数游戏

  • 需求

  1. 程序启动时，系统将自动生成一个1~100之间的随机整数；

  2. 之后，要求用户尝试猜测这个随机数，直到猜出正确结果为止；

  3. 每次用户输入一个结果后，系统需给出提示：大了、小了、对了；
